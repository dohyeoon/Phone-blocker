# Phone-blocker
설정한 횟수보다 많이 핸드폰 잠금해제를 한다면 설정한 음악이 실행하는 앱
처음 화면에는 설정한 해당 조건들을 확인할 수 있으며 위에 +버튼을 누르면 조건을 추가 할 수 있게 됨.
+버튼을 누르면 기간 및 횟수랑 앱 고유 음악 중 원하는 것과 음악을 해지하기 위한 과제를 고를 수 있는 화면이 나옴.
설정한 음악은 핸드폰이 꺼지는 것을 막기 위해 화면 자동 잠금 방지를 해야함.
설정한 음악은 최대 음향으로 틀어져야 하기해 오디오 시스템 설정보다 더 높은 우선순위를 지녀야 하기해 각 기기에 해당하는 api(제한 될 시, 하나의 os에서만 다운 받게 설정)권한을 받아 음향 조절을 못하게 해야 함
앱에서 조건를 해결하지 않고 강제 종료를 한다면 기기에서 인식하고 백그라운드에 1분마다 무한 푸쉬 알림을 예약을 해야 함.
푸쉬 알람을 누르면 바로 과제를 수행하는 화면으로 넘어감.
프론트는 React native, 백엔드엔 자바,DB는 Firebase Realtime Database, Infrastructure Firestore를 사용할 예정.
